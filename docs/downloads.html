About Downloading

Using Netscape or Mozilla? If you're using Netscape for downloading, you might want to use the "Save As" feature as prompted via a right click. This is because Netscape thinks it should install jars into itself if left-clicked. Doh!

Applications for Jesktop

Applications for Jesktop are listed on the www.jesktop.net website. There are a couple of applications that come with Jesktop from the Apache site - "Explorer" and some miscellaenous apps.

Jars and Interfaces for use with other projects

If you are a developer of a GUI app and want to include jars or classes to make it Jesktop compatible.

The API for Jesktop contained applications is available here (19Kb). This is useful if you are planning to create your own Jesktop compatible applications. For your own development, it is sufficuent for you to include the contained classes you jar.

Three very simple demos that represent complete jesktop compatible applications are here (22Kb), here (22Kb) and here (22Kb). They contain source.

FrimbleDemo demonstrates the use of the abstracted Frame/JinternalFrame Frimble library. FrimbleDemo2 shows an alternative way of doing the same (perhaps simpler). FrimbleDemo3 (in progress) shows an alternative being compatible with JavaWebStart too.

The API for Jesktop's Kernel is available here (17Kb). This is useful if you are planning to create your own Jesktop aware applications. This is a deeper and optional level to that above.
